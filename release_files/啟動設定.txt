# 【模式】請填：換房　或　掛房
#    換房 = 一桌一桌巡，找名單上的人跟著下注
#    掛房 = 固定待在「目前這一桌」跟注，不換桌
#    （想先做開機自我檢查，這行改填：檢查）
#    （找不到星城視窗時，這行改填：視窗診斷）
#    （校正餘額/帳號讀取位置，這行改填：餘額測試）
模式=換房

# 【下注】請填：開　或　關
#    開 = 實戰，會真的下注
#    關 = 測試，只會辨識、開關統計表，但不下注
下注=開

# 【帳號】這台掛機的玩家帳號名稱（餘額上傳到試算表時用）。
#    直接打字輸入，不再用 OCR 辨識名字（OCR 容易讀錯）。
#    例：帳號=我的帳號名
帳號=

# 【影像辨識房間】少數暱稱（藝術字/反白）OCR 讀不準，改用「比對長相」找。
#    這裡指定這些名字固定待在哪一桌 → 只在那一桌用影像比對找它，其餘房間完全不找
#    （避免在別桌把長相相近的路人誤比中而跟錯人），也更快。
#    格式：暱稱:桌號，多個用「、」或逗號分隔。
#    留空 = 用內建預設（閃光福雷噓→No.15、三下治灸→No.14）。改房號就照下面這樣填：
影像辨識房間=閃光福雷噓:15、三下治灸:14

# ==================================================
#  ↓↓↓ 以下只有「掛房」模式會用到（換房模式可不理） ↓↓↓
# ==================================================

# 【桌號】掛房要固定待著的桌號（例：5）。
#    被踢出或當機跳回大廳時，程式會自動回到這一桌。
#    留空或填 0 = 不指定（待在當前桌；回大廳時回任一桌）。
桌號=

# 【對象】掛房要跟注的對象暱稱，可填多個，用「、」或逗號分隔。
#    例：對象=阿明、小華、大雄
#    留空 = 沿用 follow_list.json 的名單。
#    若「指定的對象全部都不在這桌」→ 程式會暫停跟注（不補注防踢、
#    被踢回大廳也不會自動回桌），並透過 Telegram 通知（需先設定 bot）。
對象=
