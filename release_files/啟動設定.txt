# 【模式】請填：換房　或　掛房
#    換房 = 一桌一桌巡，找名單上的人跟著下注
#    掛房 = 固定待在「目前這一桌」跟注，不換桌
#    （想先做開機自我檢查，這行改填：檢查）
#    （找不到星城視窗時，這行改填：視窗診斷）
#    （校正餘額/帳號讀取位置，這行改填：餘額測試）
模式=換房

# 【下注】請填：開　或　關
#    開 = 實戰，會真的下注
#    關 = 測試，只會辨識、開關統計表，但不下注
下注=開
